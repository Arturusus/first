Алгоритм сортировки выбором - это Алгоритм простой алгоритм сортировки, работающий по следующему принципу:
1.На каждом этапе проходится по ещё не отсортированной части массива и находит наименьший (или наибольший) элемент.
2.Найденный элемент перемещается в начало отсортированной части массива путём обмена с первым элементом текущей неподсортированной области.
3.Этот процесс продолжается до тех пор, пока вся последовательность не станет упорядоченной.
